# Reviews Module

Reviews Module of AirCnB mocksite

## Getting Started

````
Npm run dev-react
mongod
mongo --host 127.0.0.1:27017
npm install
npm run start

npm run seed (seed current database)
`````

in chrome browser: localhost:3003/listings/0

## Running the tests

Tests are run and run with Jest
```
npm run test
```
