# Reviews Module

Reviews Module of AirCnB mocksite

## Getting Started

````
npm install
Npm run dev-react
mongod
mongo --host 127.0.0.1:27017
npm run start
`````

in chrome browser: localhost:3003/listings/0

## Seeding the DataBase

```
npm run seed
```
##Running the Tests

Tests are run and run with Jest
note: close nodemon/server before running tests
```
npm run test
```
